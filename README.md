# CSVTODB-TS

CSVTODB es un programa para estructurar arhivos csv a una base de datos.

## Comenzando 🚀

### Instalación 🔧

```
npm install
```

## Construido con 🛠️

- [Nodejs](https://nodejs.org/es/) - es un entorno de ejecución para JavaScript construido con V8
